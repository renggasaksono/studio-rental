A simple plugin to create Studio services booking.

== Description ==

Included:
* Create multiple services and price
* Create addons for each services
* Simple booking form

== Installation ==

1. Upload `studio-rental.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
